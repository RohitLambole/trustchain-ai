# TrustChain AI Frontend

Next.js 15 banking cybersecurity dashboard for customers, security analysts, and admins.

## Run

```bash
npm install
npm run dev
```

## Pages

- `/login`
- `/register`
- `/totp`
- `/forgot-password`
- `/dashboard`
- `/analyst`
- `/admin`
- `/devices`
- `/alerts`
- `/audit`

## Integrations

- Backend API: `NEXT_PUBLIC_API_BASE_URL`
- ML API: `NEXT_PUBLIC_ML_API_BASE_URL`
- JWT access token protection
- Refresh-token retry logic
- Role-aware navigation
- React Query data fetching
- Zustand persisted auth state
